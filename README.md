Sentiment Analysis using NLP – Description

This task focuses on performing sentiment analysis on textual data using Natural Language Processing (NLP) techniques. Sentiment analysis is used to identify the emotion or opinion expressed in a piece of text, such as positive, negative, or neutral.

In this project, a small dataset of text sentences is created, which includes different types of opinions and reviews. The data is then processed and analyzed using Python. The TextBlob library is used to perform sentiment analysis. It calculates the polarity of each sentence and classifies it into positive, negative, or neutral sentiment.

After applying sentiment analysis, a new column is added to the dataset showing the sentiment of each text. This helps in understanding how different sentences are categorized based on their meaning. The results are further analyzed by counting the number of positive, negative, and neutral texts.

Finally, a bar chart is created to visualize the sentiment distribution. This makes it easy to understand the overall results in a graphical format.

Overall, this task demonstrates how NLP techniques can be used to analyze text data and extract meaningful insights. It is useful in real-world applications like analyzing customer reviews, social media comments, and feedback.
